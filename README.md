## Get started
1. To create new project run the command below i the directory 
   where you want to keep our project files
   npx create-expo-app@54.0.27 .
2. To reset the project 
   npm run reset-project
3. To Install dependencies run the command below
   npm install
4. To run the app directly on the device install expo go app on 
   your device run the command below
   npm run start
   and it will generate qr code. scan this qr using ur expo go app on your real device
5. For this app i am using google Fonts 
   "outfit (regular and bold)" you can find the fonts in assets/fonts
